# MJML to HTML Converter

Este es un proyecto que te permite convertir archivos MJML a archivos HTML utilizando la herramienta MJML.

## Requisitos

Asegúrate de tener Node.js y npm (Node Package Manager) instalados en tu sistema antes de utilizar este proyecto.

## Instalación

1. Clona este repositorio en tu local:

```**bash**
git clone https://github.com/TU_USUARIO/flexcontrol-mails.git
Navega al directorio del proyecto: cd flexcontrol-mails
****

Instala las dependencias del proyecto:
npm install o npm **i**

Uso
Para convertir un archivo MJML a HTML, ejecuta el siguiente comando en tu terminal:

npm run convert -- archivo.mjml -o archivo.html
Sustituye archivo.mjml por el nombre de tu archivo MJML de entrada y archivo.html por el nombre que deseas para el archivo HTML de salida.

Ejemplo:
npm run convert -- mi_correo.mjml -o mi_correo.html

El archivo convertido se generará en la misma ubicación que el archivo MJML de entrada.****
